HTML e Estrutura de Páginas Web

Agora que já entendemos como a internet e os sites funcionam, chegou o momento de conhecer a primeira tecnologia utilizada no desenvolvimento web: o HTML.

O HTML é a base de praticamente todas as páginas da internet. Ele é responsável por definir a estrutura e a organização dos conteúdos que serão exibidos para os usuários.

O que é HTML?

HTML significa HyperText Markup Language, ou Linguagem de Marcação de Hipertexto.

Diferente das linguagens de programação tradicionais, o HTML não é usado para criar algoritmos ou cálculos complexos. Sua função principal é estruturar informações em uma página web.

Importante: O HTML funciona como o esqueleto de um site, organizando todos os elementos que aparecerão na tela.

Como funciona o HTML?

O HTML utiliza elementos chamados de tags, que servem para indicar ao navegador o tipo de conteúdo que será exibido.

As tags normalmente possuem uma abertura e um fechamento. 

                    <p>Este é um parágrafo.</p>

Nesse exemplo, a tag <p> informa ao navegador que o conteúdo é um parágrafo.

Estrutura básica de uma página HTML

Toda página HTML possui uma estrutura básica semelhante à apresentada abaixo: 

                    <!DOCTYPE html>
                    <html>
                    <head>
                        <title>Minha Página</title>
                    </head>
                    <body>
                        Conteúdo da página
                    </body>
                    </html>

Função de cada parte

  • DOCTYPE: informa que o documento utiliza HTML5.
  • html: elemento principal da página.
  • head: contém configurações invisíveis ao usuário.
  • title: define o título exibido na aba do navegador.
  • body: contém todo o conteúdo visível da página.

Títulos

Os títulos são utilizados para organizar o conteúdo e criar uma hierarquia de informações. 

                    <h1>Título Principal</h1>
                    <h2>Subtítulo</h2>
                    <h3>Título Menor</h3>

O elemento <h1> representa o título mais importante da página, enquanto os demais são utilizados para subdividir o conteúdo.

Parágrafos

Os parágrafos são utilizados para exibir textos comuns. 

                    <p>Este é um parágrafo.</p>

Grande parte do conteúdo textual de um site é construída utilizando essa tag.

Listas

As listas ajudam a organizar informações de forma clara e objetiva.

Lista não ordenada

                    <ul>
                        <li>HTML</li>
                        <li>CSS</li>
                        <li>JavaScript</li>
                    </ul>

Lista ordenada

                    <ol>
                        <li>Planejar</li>
                        <li>Desenvolver</li>
                        <li>Publicar</li>
                    </ol>

Links

Os links permitem navegar entre páginas e sites diferentes. 

                    <a href="https://www.google.com">
                        Acessar Google
                    </a>

O atributo href informa o endereço que será acessado.

Imagens

As imagens tornam as páginas mais atrativas e ajudam a transmitir informações visualmente. 

                    <img src="imagem.jpg" alt="Descrição da imagem">

O atributo src define o caminho da imagem e o atributo alt fornece uma descrição para acessibilidade.

Botões

Os botões permitem que o usuário realize ações dentro de uma página. 

                    <button>Clique Aqui</button>

Eles são muito utilizados em formulários, menus e sistemas interativos.

Comentários

Comentários são trechos de texto que não aparecem para o usuário e servem apenas para organizar o código. 

                    <!-- Este é um comentário -->

Boas práticas ao escrever HTML

  • Utilizar nomes claros e organizados.
  • Manter a indentação correta do código.
  • Utilizar títulos de forma hierárquica.
  • Adicionar descrições em imagens.
  • Evitar estruturas desnecessariamente complexas.
Dica: Um código organizado facilita a manutenção e a compreensão do projeto.

HTML na prática

Grande parte dos sites existentes utiliza HTML como base estrutural.

Entretanto, apenas o HTML não é suficiente para criar páginas modernas e atrativas. Para adicionar cores, animações e estilos visuais, utilizamos o CSS.

Para adicionar interatividade e comportamentos dinâmicos, utilizamos o JavaScript.

Conclusão

O HTML é a tecnologia responsável pela estrutura das páginas web. Com ele, podemos criar títulos, textos, listas, links, imagens e diversos outros elementos que compõem um site.

Aprender HTML é o primeiro passo para quem deseja ingressar no mundo do desenvolvimento web.

Exercício

Crie uma página HTML contendo:

  • Um título principal.
  • Dois parágrafos.
  • Uma lista com três itens.
  • Um link para um site de sua escolha.

Tente executar o arquivo em seu navegador para visualizar o resultado.
