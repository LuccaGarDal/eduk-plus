Como a Internet e os Sites Funcionam

Todos os dias acessamos sites, assistimos vídeos, utilizamos redes sociais e realizamos pesquisas na internet. Entretanto, poucas pessoas sabem o que acontece nos bastidores para que essas informações cheguem até nossos computadores e celulares.

Neste módulo, vamos conhecer os conceitos básicos que tornam possível o funcionamento da internet e das páginas web.

O que é a Web?

A Web, também chamada de World Wide Web (WWW), é um sistema que permite o acesso a páginas e conteúdos através da internet.

Quando você acessa um site, está utilizando a Web para visualizar informações armazenadas em servidores espalhados pelo mundo.

Importante: Internet e Web não são a mesma coisa. A internet é a rede que conecta os dispositivos, enquanto a Web é um dos serviços que funciona sobre ela.

O que é um Site?

Um site é um conjunto de páginas acessíveis através de um endereço na internet.

Essas páginas podem conter textos, imagens, vídeos, formulários e diversos outros elementos.

Exemplos de sites

  • Portais de notícias.
  • Lojas virtuais.
  • Blogs.
  • Redes sociais.
  • Plataformas educacionais.

O que é um navegador?

O navegador é o programa utilizado para acessar páginas da internet.

Ele interpreta os arquivos recebidos dos servidores e exibe as páginas de forma visual para o usuário.

Exemplos de navegadores

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge
  • Safari
  • Opera

Como um site chega até você?

Quando você digita o endereço de um site no navegador, ocorre um processo relativamente simples:

  1. O usuário digita um endereço.
  2. O navegador localiza o servidor responsável.
  3. O servidor envia os arquivos do site.
  4. O navegador interpreta os arquivos.
  5. A página é exibida na tela.

Todo esse processo acontece em poucos segundos.

O que é um Servidor?

Um servidor é um computador especializado em armazenar e disponibilizar informações para outros dispositivos através da internet.

Quando alguém acessa um site, os arquivos necessários são enviados pelo servidor para o navegador do usuário.

Exemplo: Quando você acessa um site de notícias, os textos, imagens e demais conteúdos são carregados a partir de servidores conectados à internet.

O que é um Domínio?

O domínio é o endereço utilizado para localizar um site na internet.

Ele funciona de forma semelhante ao endereço de uma residência, permitindo encontrar um recurso específico dentro da rede.

Exemplos

  • google.com
  • wikipedia.org
  • gov.br

Front-end e Back-end

O desenvolvimento web normalmente é dividido em duas grandes áreas.

Front-end

É a parte visual do site, ou seja, tudo aquilo que o usuário consegue ver e interagir.

Botões, menus, textos, imagens e formulários fazem parte do front-end.

Back-end

É a parte responsável pelo processamento das informações e pelas regras do sistema.

O usuário normalmente não vê essa parte, mas ela é responsável por funções como autenticação, armazenamento de dados e integração com outros sistemas.

Principais tecnologias da Web

Existem três tecnologias fundamentais utilizadas na construção da maioria dos sites modernos.

HTML

Responsável pela estrutura da página.

CSS

Responsável pela aparência visual e pelo design.

JavaScript

Responsável pela interatividade e comportamento das páginas.

Resumo: HTML estrutura, CSS estiliza e JavaScript adiciona interatividade.

Sites Estáticos e Dinâmicos

Sites Estáticos

Apresentam conteúdos que raramente mudam e normalmente são compostos apenas por HTML, CSS e JavaScript.

Sites Dinâmicos

Geram conteúdo em tempo real utilizando servidores e bancos de dados.

Redes sociais, sistemas bancários e lojas virtuais são exemplos de sistemas dinâmicos.

Por que aprender Desenvolvimento Web?

O desenvolvimento web é uma das áreas mais acessíveis para quem deseja iniciar sua jornada na programação.

Com conhecimentos básicos, já é possível criar páginas, portfólios, sistemas simples e diversos outros projetos.

Além disso, existe uma grande demanda por profissionais capazes de criar e manter aplicações para a internet.

Conclusão

Neste módulo aprendemos os conceitos fundamentais sobre internet, sites, servidores, domínios e as principais tecnologias utilizadas no desenvolvimento web.

Esses conhecimentos servirão como base para os próximos módulos, onde começaremos a criar páginas utilizando HTML, CSS e JavaScript.
