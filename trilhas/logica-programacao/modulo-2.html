Variáveis, Tipos de Dados e Operadores

Após compreender os conceitos básicos de lógica de programação e algoritmos, está na hora de aprender como as informações são armazenadas e modificadas dentro de um programa. Para isso, são usados variáveis e tipos de dados e também operadores.

Esses conceitos não estão restritos à apenas uma ou duas linguagens de programação. Na verdade, basicamente todas linguagens utilizam de tais conceitos para armazenar dados, fazer cálculos e tomar decisões.

O que são variáveis?

Uma variável é um espaço reservado na memória do computador utilizado para armazenar informações de forma temporária durante a execução de um programa.

Uma variável nada mais é do que uma caixa com um nome que armazena um valor em si. Esse valor pode sofrer alteração e pode também ser usado como parâmetro para outras instruções ao longo do programa.

Exemplo: Imagine uma caixa chamada "idade". Dentro dela podemos armazenar o valor 21.

Na programação, as variáveis são utilizadas para guardar informações como nome, idade, endereço, preço, quantidade e diversas outras informações cruciais para o funcionamento do sistema.

Por que utilizar variáveis?

As variáveis também nos ajudam a reutilizar valores sem precisar ter que reescrevê-los. Caso não existissem, seria necessário inserir valores no código sempre que eles fossem solicitados, o que dificultaria o desenvolvimento.

Ao utilizar variáveis, podemos armazenar informações e reutilizá-las em diferentes partes do programa.

  • Facilitam a organização do código.
  • Permitem reutilizar informações.
  • Tornam os programas mais flexíveis.
  • Facilitam futuras alterações.

Tipos de dados

Entretanto, nem todas as informações fazem parte do mesmo tipo de dado. Há inforamções que representam texto, há aquelas que representam números e há também aquelas que apenas indicam valores verdadeiros ou falsos.

Para diferenciar essas informações, utilizamos os chamados tipos de dados.

Números Inteiros

São números sem casas decimais.

  • 10
  • 25
  • 100
  • -7

Números Decimais

São números que possuem casas decimais.

  • 3.14
  • 7.5
  • 10.99

Texto (String)

Representa palavras, frases ou qualquer conjunto de caracteres.

  • "Lucca"
  • "Olá Mundo"
  • "Programação"

Booleano

É um tipo de dado que pode assumir apenas dois valores:

  • Verdadeiro (True)
  • Falso (False)

Esse tipo é extremamente importante e está associado diretamente à verificação e às tomadas de decisões.

Operadores

Operadores são símbolos utilizados para realizar operações com valores.

Operadores Matemáticos

  • + Adição
  • - Subtração
  • * Multiplicação
  • / Divisão
Exemplo: Se uma variável possui o valor 10 e outra possui o valor 5, podemos somá-las para obter o resultado 15.

Operadores de Comparação

São utilizados para comparar valores.

  • == Igual a
  • != Diferente de
  • > Maior que
  • < Menor que

O resultado de uma comparação será sempre verdadeiro ou falso.

Aplicação prática

Imagine o sistema de uma escola que precisa armazenar as notas de um aluno e o nome desse aluno. Nesse caso, o nome do aluno deve ser armazenado como texto e as notas devem ser armazenadas como números.

Em seguida, o sistema pode utilizar operadores matemáticos e de comparação para verificar se a nota é suficiente para a aprovação do aluno.

Esse é um exemplo simples de como variáveis, tipos de dados e operadores trabalham juntos para resolver problemas reais.
